## Instalacion

Para installar Fenextjs puede usar uno de los siguientes comandos dependiendo de la paqueteria de su preferencia

##### Npm
```code copy
npm i fenextjs
```

##### Bun
```code copy
bun i fenextjs
```

##### Pnpm
```code copy
pnpm i fenextjs
```
