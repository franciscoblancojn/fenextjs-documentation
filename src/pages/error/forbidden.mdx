# ErrorForbidden

Clase que representa un error de acceso denegado (HTTP 403) en la biblioteca Fenextjs.

### Importación

Para importar el componente ErrorForbidden, se puede hacer desde fenextjs

```tsx copy
import { ErrorForbidden } from "fenextjs";
```

### Parámetros

| Parámetro | Tipo      | Requerido | Default             | Descripcion                                                         |
| --------- | --------- | --------- | ------------------- | ------------------------------------------------------------------- |
| code      | ErrorCode | no        | ErrorCode.FORBIDDEN | Código de error específico que indica que el acceso está prohibido. |
| message   | string    | no        | "Forbidden"         | Mensaje de error que describe la restricción de acceso.             |
