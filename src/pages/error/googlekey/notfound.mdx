# ErrorGoogleKeyNotFound

Clase que representa un error de clave de Google no encontrada en la biblioteca Fenextjs.

### Importacion

Para importar el componente ErrorGoogleKeyNotFound, se puede hacer desde fenextjs

```tsx copy
import { ErrorGoogleKeyNotFound } from "fenextjs";
```

### Parametros

| Parametro | Tipo      | Requerido | Default                        | Descripcion                                                                   |
| --------- | --------- | --------- | ------------------------------ | ----------------------------------------------------------------------------- |
| code      | ErrorCode | no        | ErrorCode.GOOGLE_KEY_NOT_FOUND | Código de error específico que indica que la clave de Google no se encuentra. |
| message   | string    | no        | "Google Key not Found"         | Mensaje de error que describe la ausencia de la clave de Google.              |
