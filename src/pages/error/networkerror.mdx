# ErrorNetworkError

Clase que representa un error de red en la biblioteca Fenextjs.

### Importacion

Para importar el componente ErrorNetworkError, se puede hacer desde fenextjs

```tsx copy
import { ErrorNetworkError } from "fenextjs";
```

### Parametros

| Parametro | Tipo      | Requerido | Default                 | Descripcion                                                                      |
| --------- | --------- | --------- | ----------------------- | -------------------------------------------------------------------------------- |
| code      | ErrorCode | no        | ErrorCode.NETWORK_ERROR | Código de error específico que indica un error relacionado con la red.           |
| message   | string    | no        | "Network Error"         | Mensaje de error que describe la naturaleza del problema relacionado con la red. |
