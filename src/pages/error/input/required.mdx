# ErrorInputRequired

Clase que representa un error de entrada obligatoria en la biblioteca Fenextjs.

### Importación

Para importar el componente ErrorInputRequired, se puede hacer desde fenextjs

```tsx copy
import { ErrorInputRequired } from "fenextjs";
```

### Parámetros

| Parámetro | Tipo      | Requerido | Default                  | Descripcion                                                                                          |
| --------- | --------- | --------- | ------------------------ | ---------------------------------------------------------------------------------------------------- |
| code      | ErrorCode | no        | ErrorCode.INPUT_REQUIRED | Código de error específico que indica que la entrada es obligatoria y no se proporcionó.             |
| message   | string    | no        | "Input Required"         | Mensaje de error que describe la naturaleza del error cuando la entrada requerida no se proporciona. |
| input     | string    | no        |                          | Texto opcional que proporciona información sobre la entrada que causó el error.                      |
