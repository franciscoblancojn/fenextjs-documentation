# ErrorInputOutOfRange

Clase que representa un error de entrada fuera de rango en la biblioteca Fenextjs.

### Importación

Para importar el componente ErrorInputOutOfRange, se puede hacer desde fenextjs

```tsx copy
import { ErrorInputOutOfRange } from "fenextjs";
```

### Parámetros

| Parámetro | Tipo | Requerido | Default | Descripcion |
| --------- | ---- | --------- | ------- | ----------- |
| code | ErrorCode | no | ErrorCode.INPUT_OUT_OF_RANGE | Código de error específico que indica que la entrada está fuera del rango permitido. |
| message | string | no | "Input Out of Range" | Mensaje de error que describe la naturaleza del error cuando la entrada está fuera del rango permitido. Si se proporciona un rango específico, el mensaje lo incluirá. |
| input | string | no |  | Texto opcional que proporciona información sobre la entrada que causó el error. |
| range | number[] | no |  | Rango opcional que especifica los valores permitidos para la entrada. El mensaje incluirá el rango si se proporciona. |
