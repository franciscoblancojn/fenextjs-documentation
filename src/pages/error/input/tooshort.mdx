# ErrorInputTooShort

Clase que representa un error de entrada demasiado corta en la biblioteca Fenextjs.

### Importación

Para importar el componente ErrorInputTooShort, se puede hacer desde fenextjs

```tsx copy
import { ErrorInputTooShort } from "fenextjs";
```

### Parámetros

| Parámetro | Tipo           | Requerido | Default                   | Descripcion                                                                                                                                                                       |
| --------- | -------------- | --------- | ------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| code      | ErrorCode      | no        | ErrorCode.INPUT_TOO_SHORT | Código de error específico que indica que la entrada es demasiado corta.                                                                                                          |
| message   | string         | no        | "Input Too Short"         | Mensaje de error que describe la naturaleza del error cuando la entrada es más corta que la longitud mínima permitida. Si se proporciona un valor mínimo, el mensaje lo incluirá. |
| input     | string         | no        |                           | Texto opcional que proporciona información sobre la entrada que causó el error.                                                                                                   |
| min       | number \| Date | no        |                           | Valor opcional que especifica la longitud mínima permitida para la entrada. El mensaje incluirá este valor si se proporciona.                                                     |
