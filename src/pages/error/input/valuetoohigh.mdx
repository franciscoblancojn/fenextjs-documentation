# ErrorInputValueTooHigh

Clase que representa un error de valor de entrada demasiado alto en la biblioteca Fenextjs.

### Importación

Para importar el componente ErrorInputValueTooHigh, se puede hacer desde fenextjs

```tsx copy
import { ErrorInputValueTooHigh } from "fenextjs";
```

### Parámetros

| Parámetro | Tipo | Requerido | Default | Descripcion |
| --------- | ---- | --------- | ------- | ----------- |
| code | ErrorCode | no | ErrorCode.INPUT_VALUE_TOO_HIGH | Código de error específico que indica que el valor de la entrada es demasiado alto. |
| message | string | no | "Input Value Too High" | Mensaje de error que describe la naturaleza del error cuando el valor de la entrada excede el valor máximo permitido. Si se proporciona un valor máximo, el mensaje lo incluirá. |
| input | string | no |  | Texto opcional que proporciona información sobre la entrada que causó el error. |
| max | number \| Date | no |  | Valor opcional que especifica el valor máximo permitido para la entrada. El mensaje incluirá este valor si se proporciona. |
