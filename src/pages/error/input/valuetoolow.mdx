# ErrorInputValueTooLow

Clase que representa un error de valor de entrada demasiado bajo en la biblioteca Fenextjs.

### Importación

Para importar el componente ErrorInputValueTooLow, se puede hacer desde fenextjs

```tsx copy
import { ErrorInputValueTooLow } from "fenextjs";
```

### Parámetros

| Parámetro | Tipo           | Requerido | Default                       | Descripcion                                                                                                                                                                           |
| --------- | -------------- | --------- | ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| code      | ErrorCode      | no        | ErrorCode.INPUT_VALUE_TOO_LOW | Código de error específico que indica que el valor de la entrada es demasiado bajo.                                                                                                   |
| message   | string         | no        | "Input Value Too Low"         | Mensaje de error que describe la naturaleza del error cuando el valor de la entrada es inferior al valor mínimo permitido. Si se proporciona un valor mínimo, el mensaje lo incluirá. |
| input     | string         | no        |                               | Texto opcional que proporciona información sobre la entrada que causó el error.                                                                                                       |
| min       | number \| Date | no        |                               | Valor opcional que especifica el valor mínimo permitido para la entrada. El mensaje incluirá este valor si se proporciona.                                                            |
