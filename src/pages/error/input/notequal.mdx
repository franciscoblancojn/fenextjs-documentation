# ErrorInputNotEqual

Clase que representa un error de entrada que no es igual al valor esperado en la biblioteca Fenextjs.

### Importación

Para importar el componente ErrorInputNotEqual, se puede hacer desde fenextjs

```tsx copy
import { ErrorInputNotEqual } from "fenextjs";
```

### Parámetros

| Parámetro | Tipo | Requerido | Default | Descripcion |
| --------- | ---- | --------- | ------- | ----------- |
| code | ErrorCode | no | ErrorCode.INPUT_NOT_EQUAL | Código de error específico que indica que la entrada no es igual al valor esperado. |
| message | string | no | "Not Equal Input" | Mensaje de error que describe la naturaleza del error cuando la entrada no es igual al valor esperado. Si se proporciona un valor específico con el cual se espera la igualdad, el mensaje lo incluirá. |
| input | string | no |  | Texto opcional que proporciona información sobre la entrada que causó el error. |
| equal | any | no |  | Valor opcional que especifica el valor con el cual se esperaba que la entrada fuera igual. |
