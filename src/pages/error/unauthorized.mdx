# ErrorUnauthorized

Clase que representa un error de acceso no autorizado (HTTP 401) en la biblioteca Fenextjs.

### Importación

Para importar el componente ErrorUnauthorized, se puede hacer desde fenextjs

```tsx copy
import { ErrorUnauthorized } from "fenextjs";
```

### Parámetros

| Parámetro | Tipo      | Requerido | Default                | Descripcion                                                                                                         |
| --------- | --------- | --------- | ---------------------- | ------------------------------------------------------------------------------------------------------------------- |
| code      | ErrorCode | no        | ErrorCode.UNAUTHORIZED | Código de error específico que indica que el usuario no está autorizado para realizar la acción solicitada.         |
| message   | string    | no        | "Unauthorized"         | Mensaje de error que describe la naturaleza del problema cuando el acceso está prohibido por falta de autorización. |
