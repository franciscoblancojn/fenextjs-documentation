# ErrorMethodNotAllowed

Clase que representa un error de método no permitido (HTTP 405) en la biblioteca Fenextjs.

### Importación

Para importar el componente ErrorMethodNotAllowed, se puede hacer desde fenextjs

```tsx copy
import { ErrorMethodNotAllowed } from "fenextjs";
```

### Parámetros

| Parámetro | Tipo | Requerido | Default | Descripcion |
| --------- | ---- | --------- | ------- | ----------- |
| code | ErrorCode | no | ErrorCode.METHOD_NOT_ALLOWED | Código de error específico que indica que el método HTTP utilizado no está permitido. |
| message | string | no | "Method Not Allowed" | Mensaje de error que describe la naturaleza del problema cuando se intenta usar un método HTTP no permitido. |
