# ErrorBadRequest

Clase que representa un error de solicitud incorrecta (HTTP 400) en la biblioteca Fenextjs.

### Importación

Para importar el componente ErrorBadRequest, se puede hacer desde fenextjs

```tsx copy
import { ErrorBadRequest } from "fenextjs";
```

### Parámetros

| Parámetro | Tipo | Requerido | Default | Descripcion |
| --------- | ---- | --------- | ------- | ----------- |
| code | ErrorCode | no | ErrorCode.BAD_REQUEST | Código de error específico que representa una solicitud incorrecta. |
| message | string | no | "Bad Request" | Mensaje de error que describe la naturaleza del problema. |
