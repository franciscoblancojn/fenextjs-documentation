# ErrorFileNotFound

Clase que representa un error de archivo no encontrado en la biblioteca Fenextjs.

### Importación

Para importar el componente ErrorFileNotFound, se puede hacer desde fenextjs

```tsx copy
import { ErrorFileNotFound } from "fenextjs";
```

### Parámetros

| Parámetro | Tipo | Requerido | Default | Descripcion |
| --------- | ---- | --------- | ------- | ----------- |
| code | ErrorCode | no | ErrorCode.FILE_NOT_FOUND | Código de error específico que indica que el archivo no se encontró. |
| message | string | no | "File not Found" | Mensaje de error que describe la ausencia del archivo solicitado. |
