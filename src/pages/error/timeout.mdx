# ErrorTimeout

Clase que representa un error de tiempo de espera en la solicitud (HTTP 408) en la biblioteca Fenextjs.

### Importación

Para importar el componente ErrorTimeout, se puede hacer desde fenextjs

```tsx copy
import { ErrorTimeout } from "fenextjs";
```

### Parámetros

| Parámetro | Tipo      | Requerido | Default           | Descripcion                                                                                                      |
| --------- | --------- | --------- | ----------------- | ---------------------------------------------------------------------------------------------------------------- |
| code      | ErrorCode | no        | ErrorCode.TIMEOUT | Código de error específico que indica que la solicitud ha excedido el tiempo de espera permitido.                |
| message   | string    | no        | "Request Timeout" | Mensaje de error que describe la naturaleza del problema cuando se excede el tiempo de espera para la solicitud. |
