# Card_Enum

Enumeración que define los diferentes tipos de tarjetas de crédito y débito reconocidas.

### Importación

Para importar el componente Card_Enum, se puede hacer desde fenextjs

```tsx copy
import { Card_Enum } from "fenextjs";
```

### Card_Enum

Enumeración que contiene las opciones para los tipos de tarjeta reconocidos.

| key                  | value                  | description                                    |
| -------------------- | ---------------------- | ---------------------------------------------- |
| VISA                 | 'VISA'                 | Tarjeta Visa.                                  |
| MASTERCARD           | 'MASTERCARD'           | Tarjeta Mastercard.                            |
| AMEX                 | 'AMEX'                 | Tarjeta American Express.                      |
| DISCOVER             | 'DISCOVER'             | Tarjeta Discover.                              |
| DINERS               | 'DINERS'               | Tarjeta Diners Club.                           |
| DINERS_CARTE_BLANCHE | 'DINERS_CARTE_BLANCHE' | Tarjeta Diners Club Carte Blanche.             |
| JCB                  | 'JCB'                  | Tarjeta JCB.                                   |
| VISA_ELECTRON        | 'VISA_ELECTRON'        | Tarjeta Visa Electron.                         |
| OTHER                | 'OTHER'                | Tipo de tarjeta desconocido o no especificado. |
