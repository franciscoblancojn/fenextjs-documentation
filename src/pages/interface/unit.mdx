# UnitProps

Enumeraciones que definen las unidades de medida para distancia, peso y volumen.

### Importación

Para importar el componente UnitProps, se puede hacer desde fenextjs

```tsx copy
import { UnitProps } from "fenextjs";
```

### Unit_Distance

Enumeración que define las unidades de medida para distancia.

| key | value | description  |
| --- | ----- | ------------ |
| MM  | 'MM'  | Milímetros.  |
| CM  | 'CM'  | Centímetros. |
| M   | 'M'   | Metros.      |
| IN  | 'IN'  | Pulgadas.    |
| FT  | 'FT'  | Pies.        |

### Unit_Weight

Enumeración que define las unidades de medida para peso.

| key | value | description |
| --- | ----- | ----------- |
| G   | 'G'   | Gramos.     |
| KG  | 'KG'  | Kilogramos. |
| OZ  | 'OZ'  | Onzas.      |
| LB  | 'LB'  | Libras.     |

### Unit_Volumen

Enumeración que define las unidades de medida para volumen.

| key | value | description          |
| --- | ----- | -------------------- |
| ML  | 'ML'  | Mililitros.          |
| L   | 'L'   | Litros.              |
| CC  | 'CC'  | Centímetros cúbicos. |
| CM3 | 'CM3' | Centímetros cúbicos. |
| M3  | 'M3'  | Metros cúbicos.      |
| IN3 | 'IN3' | Pulgadas cúbicas.    |
| FT3 | 'FT3' | Pies cúbicos.        |
| GAL | 'GAL' | Galones.             |
