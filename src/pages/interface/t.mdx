# \_TProps

Interfaz que define propiedades opcionales para el uso de una función personalizada de traducción.

### Importación

Para importar el componente \_TProps, se puede hacer desde fenextjs

```tsx copy
import { _TProps } from "fenextjs";
```

### Parámetros

| Parámetro | Tipo                   | Requerido | Default   | Descripcion                                                                                     |
| --------- | ---------------------- | --------- | --------- | ----------------------------------------------------------------------------------------------- |
| useT      | boolean (opcional)     | no        | undefined | Indica si se debe utilizar la función de traducción personalizada `_t`.                         |
| \_t       | \_TFunciton (opcional) | no        | undefined | Función de traducción personalizada que recibe un string y devuelve un valor de cualquier tipo. |

### \_TFunciton

Tipo de función que define una función de traducción personalizada.

| key                 | value   | description                                                         |
| ------------------- | ------- | ------------------------------------------------------------------- |
| (d: string) =\> any | Función | Función que recibe un string y devuelve un valor de cualquier tipo. |
