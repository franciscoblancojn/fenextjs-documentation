# ImgDataProps

Interfaz que define las propiedades para configurar la imagen base en el componente Img.

### Importación

Para importar el componente ImgDataProps, se puede hacer desde fenextjs

```tsx copy
import { ImgDataProps } from "fenextjs";
```

### Parámetros

| Parámetro        | Tipo                        | Requerido | Default   | Descripcion                                                        |
| ---------------- | --------------------------- | --------- | --------- | ------------------------------------------------------------------ |
| id               | string \| number (opcional) | no        | undefined | ID de la imagen.                                                   |
| name             | string (opcional)           | no        | undefined | Nombre de la imagen.                                               |
| alt              | string (opcional)           | no        | undefined | Texto alternativo de la imagen.                                    |
| src              | string                      | no        | N/A       | URL de la imagen.                                                  |
| srcMin1920       | string (opcional)           | no        | undefined | URL de la imagen para pantallas de 1920px o mayores.               |
| srcMin1680       | string (opcional)           | no        | undefined | URL de la imagen para pantallas de 1680px o mayores.               |
| srcMin1440       | string (opcional)           | no        | undefined | URL de la imagen para pantallas de 1440px o mayores.               |
| srcMin1024       | string (opcional)           | no        | undefined | URL de la imagen para pantallas de 1024px o mayores.               |
| srcMin992        | string (opcional)           | no        | undefined | URL de la imagen para pantallas de 992px o mayores.                |
| srcMin768        | string (opcional)           | no        | undefined | URL de la imagen para pantallas de 768px o mayores.                |
| srcMin575        | string (opcional)           | no        | undefined | URL de la imagen para pantallas de 575px o mayores.                |
| srcThumbnail_200 | string (opcional)           | no        | undefined | URL de la imagen en miniatura para tamaños de 200x200px o mayores. |
| srcThumbnail_100 | string (opcional)           | no        | undefined | URL de la imagen en miniatura para tamaños de 100x100px o mayores. |
