# AlertProps

Interfaz que define las propiedades para configurar una alerta.

### Importación

Para importar el componente AlertProps, se puede hacer desde fenextjs

```tsx copy
import { AlertProps } from "fenextjs";
```

### Parámetros

| Parámetro | Tipo | Requerido | Default | Descripcion |
| --------- | ---- | --------- | ------- | ----------- |
| message | string | no | N/A | Mensaje de la alerta que se mostrará al usuario. |
| type | AlertType \| keyof typeof AlertType | no | N/A | Especifica el tipo de alerta; acepta un valor del enum `AlertType` o una cadena equivalente. |
| data | T (opcional) | no | undefined | Datos adicionales opcionales asociados con la alerta. |

### AlertType

Enumeración que define los diferentes tipos de alerta disponibles.

| key | value | description |
| --- | --- | --- |
| OK | 'OK' | Tipo de alerta que indica que la operación fue éxitosa. |
| ERROR | 'ERROR' | Tipo de alerta que indica un error. |
| NORMAL | 'NORMAL' | Tipo de alerta que representa un estado informativo o neutral. |
| WARNING | 'WARNING' | Tipo de alerta que indica una advertencia. |
