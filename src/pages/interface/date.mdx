# Date

Tipos de fechas y días de la semana utilizados para la configuración de fechas y calendarios.

### Importación

Para importar el componente Date, se puede hacer desde fenextjs

```tsx copy
import { Date } from "fenextjs";
```


### TypeDate

Tipo que define los valores posibles para configurar una selección de fecha.

| key | value | description |
| --- | --- | --- |
| date | 'date' | Permite la selección de una fecha específica. |
| month | 'month' | Permite la selección de un mes específico. |
| week | 'week' | Permite la selección de una semana específica. |
| time | 'time' | Permite la selección de una hora específica. |

### DaysEnum

Enumeración que define los días de la semana.

| key | value | description |
| --- | --- | --- |
| Monday | 'Monday' | Representa el día lunes. |
| Tuesday | 'Tuesday' | Representa el día martes. |
| Wednesday | 'Wednesday' | Representa el día miércoles. |
| Thursday | 'Thursday' | Representa el día jueves. |
| Friday | 'Friday' | Representa el día viernes. |
| Saturday | 'Saturday' | Representa el día sábado. |
| Sunday | 'Sunday' | Representa el día domingo. |
