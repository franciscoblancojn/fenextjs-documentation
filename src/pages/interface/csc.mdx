# CSCProps

Interfaz que define las propiedades relacionadas con un país, estado y ciudad, junto con versiones de texto de cada una.

### Importación

Para importar el componente CSCProps, se puede hacer desde fenextjs

```tsx copy
import { CSCProps } from "fenextjs";
```

### Parámetros

| Parámetro | Tipo | Requerido | Default | Descripcion |
| --------- | ---- | --------- | ------- | ----------- |
| country | CountryProps (opcional) | no | undefined | Propiedades del país seleccionadas, definidas en la interfaz `CountryProps`. |
| state | StateProps (opcional) | no | undefined | Propiedades del estado seleccionado, definidas en la interfaz `StateProps`. |
| city | CityProps (opcional) | no | undefined | Propiedades de la ciudad seleccionada, definidas en la interfaz `CityProps`. |

### CountryProps

Interfaz que define las propiedades de un país.

| id | type | required | default | description |
| --- | --- | --- | --- | --- |
| id | number | true | N/A | ID único del país. |
| text | string | true | N/A | Nombre del país. |
| code | string | true | N/A | Código ISO del país. |
| img | string (opcional) | false | undefined | URL de la imagen representativa del país. |
| code_phone | string (opcional) | false | undefined | Código telefónico del país. |
| lang | string (opcional) | false | undefined | Código idioma del país. |

### StateProps

Interfaz que define las propiedades de un estado.

| id | type | required | default | description |
| --- | --- | --- | --- | --- |
| id | number | true | N/A | ID único del estado. |
| id_country | number | true | N/A | ID del país al que pertenece el estado. |
| text | string | true | N/A | Nombre del estado. |

### CityProps

Interfaz que define las propiedades de una ciudad.

| id | type | required | default | description |
| --- | --- | --- | --- | --- |
| id | number | true | N/A | ID único de la ciudad. |
| id_state | number | true | N/A | ID del estado al que pertenece la ciudad. |
| id_country | number | true | N/A | ID del país al que pertenece la ciudad. |
| text | string | true | N/A | Nombre de la ciudad. |
| nameAve | string (opcional) | false | undefined | Nombre de la avenida principal de la ciudad. |

### CSCStringProps

Interfaz alternativa que define los valores de país, estado y ciudad como cadenas de texto.

| id | type | required | default | description |
| --- | --- | --- | --- | --- |
| country | string (opcional) | false | undefined | Nombre del país en formato de texto. |
| state | string (opcional) | false | undefined | Nombre del estado en formato de texto. |
| city | string (opcional) | false | undefined | Nombre de la ciudad en formato de texto. |
