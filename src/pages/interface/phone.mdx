# PhoneProps

Interfaz que define las propiedades de un número de teléfono, incluyendo el código de país y la bandera.

### Importación

Para importar el componente PhoneProps, se puede hacer desde fenextjs

```tsx copy
import { PhoneProps } from "fenextjs";
```

### Parámetros

| Parámetro | Tipo | Requerido | Default | Descripcion |
| --------- | ---- | --------- | ------- | ----------- |
| img | string (opcional) | no | undefined | URL de la imagen de la bandera del país. |
| code | string | no | N/A | Código telefónico del país (e.g., '+1', '+44'). |
| code_country | string (opcional) | no | undefined | Código del país en formato ISO (e.g., 'US', 'GB'). |
| country | CountryProps (opcional) | no | undefined | Propiedades adicionales del país, definidas por la interfaz `CountryProps`. |
| number | string | no | N/A | Número de teléfono. |
| tel | string (opcional) | no | undefined | Número de teléfono en formato 'tel' para enlaces de marcación rápida. |

### PhoneCodeProps

Interfaz que define las propiedades relacionadas con el código telefónico de un país.

| key | value | description |
| --- | --- | --- |
| img | string (opcional) | URL de la imagen de la bandera del país. |
| code | string | Código telefónico del país. |
| code_country | string (opcional) | Código del país en formato ISO. |
| country | CountryProps (opcional) | Propiedades adicionales del país. |
