# Imagenes

Fenextjs posee Imagenes para implementar en Nextjs con el objetivo de agilizar el desarrollo y mantener un estandar.

#### **Lista de Imagenes disponibles**

- **[_Placeholder_](/img-placeholder/placeholder)**
- **[_User_](/img-placeholder/user)**

### Instalacion independiente

En caso de que solo quieras usar la parte de Imagenes de Fenextjs sin installar el resto de funcionalidades, puedes hacerlo de siguiente manera:

##### Npm

```code copy
npm i fenextjs-img-placeholder
```

##### Bun

```code copy
bun i fenextjs-img-placeholder
```

##### Pnpm

```code copy
pnpm i fenextjs-img-placeholder
```
