# Errores

Fenextjs posee Errores para implementar en Nextjs con el objetivo de mantener un estandar.

### Instalacion independiente

En caso de que solo quieras usar la parte de Errores de Fenextjs sin installar el resto de funcionalidades, puedes hacerlo de siguiente manera:

##### Npm

```code copy
npm i fenextjs-error
```

##### Bun

```code copy
bun i fenextjs-error
```

##### Pnpm

```code copy
pnpm i fenextjs-error
```
