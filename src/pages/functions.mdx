# Funciones

Fenextjs posee Funciones para implementar en Nextjs con el objetivo de agilizar el desarrollo.

#### **Lista de Funciones disponibles**

- **[_\_t_](/functions/t)**

### Instalacion independiente

En caso de que solo quieras usar la parte de Funciones de Fenextjs sin installar el resto de funcionalidades, puedes hacerlo de siguiente manera:

##### Npm

```code copy
npm i fenextjs-functions
```

##### Bun

```code copy
bun i fenextjs-functions
```

##### Pnpm

```code copy
pnpm i fenextjs-functions
```
