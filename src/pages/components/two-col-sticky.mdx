# TwoColSticky

El componente TwoColSticky es un componente que muestra un dos columnas con una de ellas sticky.

### Ejemplo:

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=twocolsticky-twocolsticky--index&viewMode=story" />

### Importación

Para importar el componente TwoColSticky, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import {
  TwoColSticky,
  TwoColStickyProps,
} from "fenextjs/cjs/component/TwoColSticky";
```

o

```tsx copy
import {
  TwoColSticky,
  TwoColStickyProps,
} from "fenextjs-component/cjs/TwoColSticky";
```

### Parámetros:

| Parámetro |        Tipo         | Requerido | Default | Descripción                                                  |
| --------- | :-----------------: | :-------: | :-----: | :----------------------------------------------------------- |
| \_t       |      `string`       |    No     |         | Propiedad de internacionalización.                           |
| children  |     `ReactNode`     |    No     |         | Contenido principal del componente.                          |
| colSticky |     `ReactNode`     |    No     |         | Contenido de la columna que se mantiene pegajosa (sticky).   |
| posCol    | `"left" \| "right"` |    No     | "left"  | Posición de la columna pegajosa. Puede ser "left" o "right". |
