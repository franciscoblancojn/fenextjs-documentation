# ImgSlider

El componente ImgSlider es un componente que muestra un slider de imagenes.

### Ejemplo:

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=img-imgslider--index&viewMode=story" />

### Importación

Para importar el componente ImgSlider, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import { ImgSlider, ImgSliderProps } from "fenextjs/cjs/component/ImgSlider";
```

o

```tsx copy
import { ImgSlider, ImgSliderProps } from "fenextjs-component/cjs/ImgSlider";
```

### Parámetros:

| Parámetro |         Tipo          | Requerido | Default | Descripción                                   |
| --------- | :-------------------: | :-------: | :-----: | :-------------------------------------------- |
| imgs      |     `ImgProps[]`      |    Sí     |         | Lista de imágenes.                            |
| step      |       `number`        |    No     |         | Paso de la imagen.                            |
| setStep   | `(e: number) => void` |    No     |         | Función para establecer el paso de la imagen. |
