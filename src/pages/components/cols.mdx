# Cols

El componente Cols es un componente que muestra el contenido dividido en columnas.

### Ejemplo

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=cols-cols--index&viewMode=story" />

### Importacion

Para importar el componente Cols, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import { Cols, ColsProps } from "fenextjs/cjs/component/Cols";
```

o

```tsx copy
import { Cols, ColsProps } from "fenextjs-component/cjs/Cols";
```

### Parametros

| Propiedad   |       Tipo       | Requerido | Default | Descripción                                                     |
| ----------- | :--------------: | :-------: | :-----: | :-------------------------------------------------------------- |
| nCols       | number \| string |    no     |         | El número de columnas para el componente.                       |
| breakInside |     boolean      |    no     |         | Indica si se permite el salto de columna dentro del componente. |
| children    |    ReactNode     |    no     |         | Contenido del componente.                                       |

### Storybook

Para ver el storybook del componente lo puede hacer con este [link](https://fenextjs-component-storybook.vercel.app/?path=/story/cols-cols--index)

### Usos

- Uso básico del componente Cols

```tsx copy
<Cols
  nCols={3}
  breakInside={true}
>
  {/* Contenido de las columnas */}
</Cols>
```
