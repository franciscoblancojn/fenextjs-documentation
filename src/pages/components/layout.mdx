# LayoutGridMenuLeft

El componente LayoutGridMenuLeft es un componente que muestra un layout con un menu a la izquierda.

### Ejemplo:

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?id=layout-layoutgridmenuleft--index&viewMode=story" />

### Importación

Para importar el componente LayoutGridMenuLeft, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import {
  LayoutGridMenuLeft,
  LayoutGridMenuLeftProps,
} from "fenextjs/cjs/component/Layout/Grid/GridMenuLeft";
```

o

```tsx copy
import {
  LayoutGridMenuLeft,
  LayoutGridMenuLeftProps,
} from "fenextjs-component/cjs/Layout/Grid/GridMenuLeft";
```

### Parámetros:

| Parámetro           |    Tipo     | Requerido | Default | Descripción                                                   |
| ------------------- | :---------: | :-------: | :-----: | :------------------------------------------------------------ |
| loader              |  `boolean`  |    No     |         | Indica si la página está actualmente en estado de carga.      |
| menuLeft            | `ReactNode` |    No     |         | Menú izquierdo del diseño.                                    |
| menuLeftActive      |  `boolean`  |    No     |         | Indica si el menú izquierdo está activo en el diseño.         |
| menuLeftMovilActive |  `boolean`  |    No     |         | Indica si el menú izquierdo móvil está activo en el diseño.   |
| useHeaderButtonMenu |  `boolean`  |    No     |         | Utilizar botón de menú en el encabezado para mostrar el menú. |
| usePageProgress     |  `boolean`  |    No     |         | Utilizar barra de progreso de página.                         |

```tsx copy

    <>
      <h3>It is Menu Left</h3>
    </>
  }
>
  <>
    <h1>It is Body</h1>
    <p>
      Lorem ipsum dolor, sit amet consectetur adipisicing elit. Aperiam

  </>

```
