# Tooltip

El componente Tooltip es un componente que muestra un contenerdor que al hacer hover muestra un contenido personalizado.

### Ejemplo:

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=tooltip-tooltip--index&viewMode=story" />

### Importación

Para importar el componente Tooltip, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import { Tooltip, TooltipProps } from "fenextjs/cjs/component/Tooltip";
```

o

```tsx copy
import { Tooltip, TooltipProps } from "fenextjs-component/cjs/Tooltip";
```

### Parámetros:

| Parámetro |              Tipo               | Requerido | Default  | Descripción                                                            |
| --------- | :-----------------------------: | :-------: | :------: | :--------------------------------------------------------------------- |
| \_t       |            `string`             |    No     |          | Propiedad de internacionalización.                                     |
| children  |           `ReactNode`           |    No     |          | Contenido del componente que activa el tooltip.                        |
| tooltip   |           `ReactNode`           |    No     |          | Contenido del tooltip a mostrar.                                       |
| positionX | `"center" \| "right" \| "left"` |    No     | "center" | Posición horizontal del tooltip. Puede ser "center", "right" o "left". |
| positionY | `"center" \| "top" \| "bottom"` |    No     |  "top"   | Posición vertical del tooltip. Puede ser "center", "top" o "bottom".   |
