# Loader

El componente Loader es un componente que muestra un loader simple.

### Ejemplo:

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=loader-loader--index&viewMode=story" />

### Importación

Para importar el componente Loader, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import { Loader, LoaderProps } from "fenextjs/cjs/component/Loader";
```

o

```tsx copy
import { Loader, LoaderProps } from "fenextjs-component/cjs/Loader";
```

### Parámetros:

| Parámetro       |   Tipo   | Requerido | Default | Descripción                     |
| --------------- | :------: | :-------: | :-----: | :------------------------------ |
| classNameLoader | `string` |    No     |         | Clase personalizada del loader. |

### Storybook

Para ver el storybook del componente lo puede hacer con este [link](https://fenextjs-component-storybook.vercel.app/?path=/story/loader-loader--index)

### Usos

- Uso básico del componente Loader

```tsx copy
<Loader />
```
