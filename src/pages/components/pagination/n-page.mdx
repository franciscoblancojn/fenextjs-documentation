# PaginationNPage

El componente PaginationNPage es un componente que muestra una paginacion con solo el selector de numero de items por pagina.

### Ejemplo

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=pagination-paginationnpage--index&viewMode=story" />

### Importacion

Para importar el componente PaginationNPage, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import {
  PaginationNPage,
  PaginationNPageProps,
} from "fenextjs/cjs/component/Pagination/PaginationNPage";
```

o

```tsx copy
import {
  PaginationNPage,
  PaginationNPageProps,
} from "fenextjs-component/cjs/Pagination/PaginationNPage";
```

### Parametros

| Parametro     |                 Tipo                  | Requerido | Default | Descripcion                                                                       |
| ------------- | :-----------------------------------: | :-------: | :-----: | :-------------------------------------------------------------------------------- |
| \_t           |               `string`                |    No     |         | Propiedad de internacionalización.                                                |
| listNpage     |   `InputSelectBaseProps['options']`   |    No     |         | Lista de opciones para seleccionar el número de páginas.                          |
| onChangeNPage |  `InputSelectBaseProps['onChange']`   |    No     |         | Función de devolución de llamada cuando cambia el número de páginas seleccionado. |
| value         |    `InputSelectBaseProps['value']`    |    No     |         | Valor seleccionado actualmente del número de páginas.                             |
| disabled      |  `InputSelectBaseProps['disabled']`   |    No     |         | Indica si el componente está deshabilitado.                                       |
| className     |  `InputSelectBaseProps['className']`  |    No     |         | Clase personalizada para el componente.                                           |
| placeholder   | `InputSelectBaseProps['placeholder']` |    No     |         | Texto de marcador de posición para el componente.                                 |
