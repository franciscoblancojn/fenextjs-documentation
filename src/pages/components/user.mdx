# TwoColSticky

El componente TwoColSticky es un componente que muestra un dos columnas con una de ellas sticky.

### Ejemplo:

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=user-user--index&viewMode=story" />

### Importación

Para importar el componente TwoColSticky, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import {
  TwoColSticky,
  TwoColStickyProps,
} from "fenextjs/cjs/component/TwoColSticky";
```

o

```tsx copy
import {
  TwoColSticky,
  TwoColStickyProps,
} from "fenextjs-component/cjs/TwoColSticky";
```

### Parámetros:

| Propiedad |             Tipo              | Requerido | Descripción                                                                         |
| --------- | :---------------------------: | :-------: | :---------------------------------------------------------------------------------- |
| user      | `Partial<UserInterfaceProps>` |    Sí     | Datos parciales del usuario (`Partial<UserInterfaceProps>`).                        |
| loader    |           `boolean`           |    No     | Indica si se debe mostrar un cargador mientras se carga la información del usuario. |

#### UserInterfaceProps

| Propiedad  |                        Tipo                         | Requerido | Descripción                                                                           |
| ---------- | :-------------------------------------------------: | :-------: | :------------------------------------------------------------------------------------ |
| status     |                  `UserStatusProps`                  |    Sí     | Estado del usuario. Puede ser VERIFY, NOVERIFY, BAN o PENDING.                        |
| id         |                      `string`                       |    Sí     | Identificador único del usuario.                                                      |
| token      |                      `string`                       |    Sí     | Token asociado al usuario.                                                            |
| name       |                      `string`                       |    Sí     | Nombre del usuario.                                                                   |
| img        |                   `ImgDataProps`                    |    Sí     | Datos de la imagen del usuario (`ImgDataProps`).                                      |
| role       |                   `UserRoleProps`                   |    Sí     | Rol del usuario. Puede ser ADMIN, CUSTOMER o BACKOFFICE.                              |
| phone      |                    `PhoneProps`                     |    No     | Datos del teléfono del usuario (`PhoneProps`).                                        |
| email      |                      `string`                       |    Sí     | Dirección de correo electrónico del usuario.                                          |
| stripe_id  |                      `string`                       |    No     | Identificador de Stripe asociado al usuario.                                          |
| dateCreate |                       `Date`                        |    Sí     | Fecha de creación del usuario.                                                        |
| verify     | `{ [id in UserTypeVerifyProps]: UserStatusProps; }` |    No     | Estado de verificación para diferentes tipos de verificación (`UserTypeVerifyProps`). |
