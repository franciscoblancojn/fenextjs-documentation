# MediaInput

El componente MediaInput es un componente que muestra un input media para gestionar archivos.

### Ejemplo

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=media-input--index&viewMode=story" />

### Importacion

Para importar el componente MediaInput, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import {
  MediaInput,
  MediaInputProps,
} from "fenextjs/cjs/component/Media/Input";
```

o

```tsx copy
import {
  MediaInput,
  MediaInputProps,
} from "fenextjs-component/cjs/Media/Input";
```

### Parametros

| Parametro         |                                Tipo                                | Requerido | Default | Descripcion                                             |
| ----------------- | :----------------------------------------------------------------: | :-------: | :-----: | :------------------------------------------------------ |
| titleView         |                              `string`                              |    No     |         | Título de la vista del componente.                      |
| textView          |                              `string`                              |    No     |         | Texto de la vista del componente.                       |
| iconView          |                            `ReactNode`                             |    No     |         | Icono de la vista del componente.                       |
| defaultValue      |                 `ImgDataProps[]` \| `ImgDataProps`                 |    No     |         | Valor predeterminado del componente.                    |
| multiple          |                             `boolean`                              |    No     |         | Indica si se permiten múltiples elementos.              |
| onChange          |   `(data: ImgDataProps[] \| ImgDataProps \| undefined) => void`    |    No     |         | Función a llamar cuando cambia el valor del componente. |
| ButtonUploadProps |                   `Omit<ButtonProps, "onClick">`                   |    No     |         | Propiedades del botón de carga para el componente.      |
| MediaPageProps    | `Omit<MediaPageProps, "onChange" \| "multiple" \| "defaultValue">` |    No     |         | Propiedades de la página de medios para el componente.  |
| ModalProps        |                 `Pick<ModalBaseBaseProps, "type">`                 |    No     |         | Propiedades del modal para el componente.               |
