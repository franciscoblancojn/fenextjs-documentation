# ErrorComponent

El componente ErrorComponent es un componente que muestra un error de ErrorFenextjs.

### Ejemplo

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=error-errorcomponent--index&viewMode=story" />

### Importacion

Para importar el componente ErrorComponent, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import {
  ErrorComponent,
  ErrorComponentProps,
} from "fenextjs/cjs/component/Error";
```

o

```tsx copy
import {
  ErrorComponent,
  ErrorComponentProps,
} from "fenextjs-component/cjs/Error";
```

### Parametros

| Propiedad |     Tipo      | Requerido | Default | Descripción                                            |
| --------- | :-----------: | :-------: | :-----: | :----------------------------------------------------- |
| error     | ErrorFenextjs |    no     |         | Error del componente.                                  |
| children  |   ReactNode   |    no     |         | Contenido para mostrar si no se usa el parametro error |

### Storybook

Para ver el storybook del componente lo puede hacer con este [link](https://fenextjs-component-storybook.vercel.app/?path=/story/error-errorcomponent--index)

### Usos

- Uso básico del componente ErrorComponent

```tsx copy
<ErrorComponent error={new ErrorFenextjs()}>
  Contenido del ErrorComponent
</ErrorComponent>
```
