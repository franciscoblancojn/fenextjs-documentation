# ProgressLine

El componente ProgressLine es un componente que muestra un progreso reprecentado en linea y porcentaje.

### Ejemplo

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=progress-progressline--index&viewMode=story" />

### Importacion

Para importar el componente ProgressLine, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import {
  ProgressLine,
  ProgressLineProps,
} from "fenextjs/cjs/component/Progress/Line";
```

o

```tsx copy
import {
  ProgressLine,
  ProgressLineProps,
} from "fenextjs-component/cjs/Progress/Line";
```

### Parametros

| Parametro |   Tipo    | Requerido | Default | Descripcion                                      |
| --------- | :-------: | :-------: | :-----: | :----------------------------------------------- |
| p         | `number`  |    Sí     |         | Número de progreso.                              |
| showP     | `boolean` |    Sí     |         | Indica si se debe mostrar el número de progreso. |

### Storybook

Para ver el storybook del componente lo puede hacer con este [link](https://fenextjs-component-storybook.vercel.app/?path=/story/progress-progressline--index)

### Usos

- Uso básico del componente ProgressLine

```tsx copy
<ProgressLine p={40} showP={true} />
```
