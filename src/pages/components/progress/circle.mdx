# ProgressCircle

El componente ProgressCircle es un componente que muestra un progreso reprecentado en circulo y porcentaje.

### Ejemplo

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=progress-progresscircle--index&viewMode=story" />

### Importacion

Para importar el componente ProgressCircle, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import {
  ProgressCircle,
  ProgressCircleProps,
} from "fenextjs/cjs/component/Progress/Circle";
```

o

```tsx copy
import {
  ProgressCircle,
  ProgressCircleProps,
} from "fenextjs-component/cjs/Progress/Circle";
```

### Parametros

| Parametro |   Tipo    | Requerido | Default | Descripcion                                      |
| --------- | :-------: | :-------: | :-----: | :----------------------------------------------- |
| p         | `number`  |    Sí     |         | Número de progreso.                              |
| showP     | `boolean` |    Sí     |         | Indica si se debe mostrar el número de progreso. |

### Storybook

Para ver el storybook del componente lo puede hacer con este [link](https://fenextjs-component-storybook.vercel.app/?path=/story/progress-progresscircle--index)

### Usos

- Uso básico del componente ProgressCircle

```tsx copy
<ProgressCircle p={40} showP={true} />
```
