# Text

El componente Text es un componente que muestra un contenedor de texto en sus distintas versiones.

### Ejemplo:

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=text-text--index&viewMode=story" />

### Importación

Para importar el componente Text, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import { Text, TextProps } from "fenextjs/cjs/component/Text";
```

o

```tsx copy
import { Text, TextProps } from "fenextjs-component/cjs/Text";
```

### Parámetros:

| Parámetro   |                                                 Tipo                                                 | Requerido | Default | Descripción                                                                                                                                       |
| ----------- | :--------------------------------------------------------------------------------------------------: | :-------: | :-----: | :------------------------------------------------------------------------------------------------------------------------------------------------ |
| \_t         |                                               `string`                                               |    No     |         | Propiedad de internacionalización.                                                                                                                |
| tag         | `"p" \| "strong" \| "small" \| "em" \| "b" \| "del" \| "i" \| "mark" \| "ins" \|   "sub"   \| "sup"` |    No     |   "p"   | Indica el tipo de etiqueta HTML a utilizar para el texto. Puede ser "p", "strong", "small", "em", "b", "del", "i", "mark", "ins", "sub", o "sup". |
| loader      |                                              `boolean`                                               |    No     |  false  | Indica si se debe mostrar un cargador mientras se carga el texto.                                                                                 |
| nLineLoader |                                               `number`                                               |    No     |    3    | Número de líneas que se deben mostrar en el cargador.                                                                                             |
