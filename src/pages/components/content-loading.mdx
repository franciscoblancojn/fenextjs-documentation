# ContentLoading

El componente ContentLoading es un componente que muestra el contenido o un loader personalizado.

### Ejemplo

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=contentloading-contentloading--index&viewMode=story" />

### Importacion

Para importar el componente ContentLoading, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import {
  ContentLoading,
  ContentLoadingProps,
} from "fenextjs/cjs/component/ContentLoading";
```

o

```tsx copy
import {
  ContentLoading,
  ContentLoadingProps,
} from "fenextjs-component/cjs/ContentLoading";
```
