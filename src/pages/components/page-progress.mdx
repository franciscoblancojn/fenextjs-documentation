# PageProgress

El componente PageProgress es un componente que muestra una barra de progreso al cargar paginas.

### Importacion

Para importar el componente PageProgress, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import {
  PageProgress,
  PageProgressProps,
} from "fenextjs/cjs/component/PageProgress";
```

o

```tsx copy
import {
  PageProgress,
  PageProgressProps,
} from "fenextjs-component/cjs/PageProgress";
```
