# ItemMenu

El componente ItemMenu es un componente que muestra un elemento del menu.

### Ejemplo:

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=header-itemmenu--index&viewMode=story" />

### Importación

Para importar el componente ItemMenu, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import {
  ItemMenu,
  ItemMenuProps,
} from "fenextjs/cjs/component/Header/ItemMenu";
```

o

```tsx copy
import {
  ItemMenu,
  ItemMenuProps,
} from "fenextjs-component/cjs/Header/ItemMenu";
```

### Parámetros:

| Parámetro     |             Tipo              | Requerido | Default | Descripción                                                          |
| ------------- | :---------------------------: | :-------: | :-----: | :------------------------------------------------------------------- |
| url           |           `string`            |    Sí     |         | URL de la página en el elemento del menú.                            |
| text          |          `ReactNode`          |    Sí     |         | Texto de la página en el elemento del menú.                          |
| icon          |          `ReactNode`          |    No     |         | Icono de la página en el elemento del menú.                          |
| subItems      | `Omit<ItemMenuProps, "_t">[]` |    No     |         | Subpáginas en el elemento del menú.                                  |
| defaultActive |           `boolean`           |    No     |         | Indica si el elemento del menú está por defecto activo para mostrar. |
