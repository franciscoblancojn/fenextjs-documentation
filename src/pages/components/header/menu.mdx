# Menu

El componente Menu es un componente que muestra un menu.

### Ejemplo

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=header-menu--index&viewMode=story" />

### Importacion

Para importar el componente Menu, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import { Menu, MenuProps } from "fenextjs/cjs/component/Header/Menu";
```

o

```tsx copy
import { Menu, MenuProps } from "fenextjs-component/cjs/Header/Menu";
```

### Parametros

| Parametro          |       Tipo        | Requerido | Default | Descripcion                                                   |
| ------------------ | :---------------: | :-------: | :-----: | :------------------------------------------------------------ |
| items              | `ItemMenuProps[]` |    No     |         | Los elementos del menú.                                       |
| className          |     `string`      |    No     |         | La clase CSS para el componente.                              |
| defaultShowSubMenu |     `boolean`     |    No     | `false` | Indica si el submenú se muestra por defecto en el componente. |

### Storybook

Para ver el storybook del componente lo puede hacer con este [link](https://fenextjs-component-storybook.vercel.app/?path=/story/header-menu--index)

### Usos

- Uso básico del componente Menu

```tsx copy
<Menu
  items={[
    {
      url: "/",
      text: "Link",
    },
  ]}
/>
```
