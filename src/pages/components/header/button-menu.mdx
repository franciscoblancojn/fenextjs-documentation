# ButtonMenu

El componente ButtonMenu es un componente que muestra un botton que permite interactuar con el menu (menu izquierdo).

### Ejemplo:

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=header-buttonmenu--index&viewMode=story" />

### Importación

Para importar el componente ButtonMenu, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import {
  ButtonMenu,
  ButtonMenuProps,
} from "fenextjs/cjs/component/Header/ButtonMenu";
```

o

```tsx copy
import {
  ButtonMenu,
  ButtonMenuProps,
} from "fenextjs-component/cjs/Header/ButtonMenu";
```

### Parámetros:

| Parámetro     |    Tipo     | Requerido | Default | Descripción                                              |
| ------------- | :---------: | :-------: | :-----: | :------------------------------------------------------- |
| loader        |  `boolean`  |    No     |         | Indica si el botón está actualmente en estado de carga.  |
| disabled      |  `boolean`  |    No     |         | Indica si el botón está desactivado o no.                |
| children      | `ReactNode` |    no     |         | Contenido del componente.                                |
| defaultActive |  `boolean`  |    No     |         | Indica si el botón está por defecto activo para mostrar. |
