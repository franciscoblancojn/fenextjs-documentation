# LoaderUser

El componente LoaderUser es un componente que muestra un loader de usuario simple.

### Ejemplo

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=loader-user--index&viewMode=story" />

### Importacion

Para importar el componente LoaderUser, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import {
  LoaderUser,
  LoaderUserProps,
} from "fenextjs/cjs/component/Loader/User";
```

o

```tsx copy
import {
  LoaderUser,
  LoaderUserProps,
} from "fenextjs-component/cjs/Loader/User";
```

### Parametros

| Parametro           |   Tipo   | Requerido | Default | Descripcion                     |
| ------------------- | :------: | :-------: | :-----: | :------------------------------ |
| classNameLoaderUser | `string` |    No     |         | Clase personalizada del loader. |
