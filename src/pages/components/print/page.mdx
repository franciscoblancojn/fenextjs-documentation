# PrintPage

El componente PrintPage es un componente que muestra para imprimir como pagina.

### Ejemplo:

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=print-page--index&viewMode=story" />

### Importación

Para importar el componente PrintPage, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import { PrintPage, PrintPageProps } from "fenextjs/cjs/component/Print/Page";
```

o

```tsx copy
import { PrintPage, PrintPageProps } from "fenextjs-component/cjs/Print/Page";
```

### Parámetros:

| Parámetro   |                       Tipo                        | Requerido | Default | Descripción                                           |
| ----------- | :-----------------------------------------------: | :-------: | :-----: | :---------------------------------------------------- |
| onComponent | `(data: PrintPageComponentProps<T>) => ReactNode` |    Sí     |         | Función que devuelve el componente para la impresión. |

### Storybook

Para ver el storybook del componente lo puede hacer con este [link](https://fenextjs-component-storybook.vercel.app/?path=/story/print-page--index)
