# Title

El componente Title es un componente que muestra un contenedor de titulo en sus distintas versiones.

### Ejemplo

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=title-title--h-1&viewMode=story" />

### Importacion

Para importar el componente Title, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import { Title, TitleProps } from "fenextjs/cjs/component/Title";
```

o

```tsx copy
import { Title, TitleProps } from "fenextjs-component/cjs/Title";
```

### Parametros

| Parametro |                           Tipo                           | Requerido | Default | Descripcion                                                                                               |
| --------- | :------------------------------------------------------: | :-------: | :-----: | :-------------------------------------------------------------------------------------------------------- |
| \_t       |                         `string`                         |    No     |         | Propiedad de internacionalización.                                                                        |
| tag       | `"h1"   \|   "h2"    \|  "h3"   \| "h4" \| "h5" \| "h6"` |    No     |  "h1"   | Indica el tipo de etiqueta HTML a utilizar para el título. Puede ser "h1", "h2", "h3", "h4", "h5" o "h6". |
| loader    |                        `boolean`                         |    No     |  false  | Indica si se debe mostrar un cargador mientras se carga el título.                                        |
