# LayoutGridMenuTop

El componente LayoutGridMenuTop es un componente que muestra un layout con un menu arriba.

### Ejemplo:

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=layout-layoutgridmenutop--index&viewMode=story" />

### Importación

Para importar el componente LayoutGridMenuTop, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import {
  LayoutGridMenuTop,
  LayoutGridMenuTopProps,
} from "fenextjs/cjs/component/Layout/Grid/GridMenuTop";
```

o

```tsx copy
import {
  LayoutGridMenuTop,
  LayoutGridMenuTopProps,
} from "fenextjs-component/cjs/Layout/Grid/GridMenuTop";
```

### Parámetros:

| Parámetro       |    Tipo     | Requerido | Default | Descripción                                              |
| --------------- | :---------: | :-------: | :-----: | :------------------------------------------------------- |
| loader          |  `boolean`  |    No     |         | Indica si la página está actualmente en estado de carga. |
| menuTop         | `ReactNode` |    No     |         | Menú superior del diseño.                                |
| usePageProgress |  `boolean`  |    No     |         | Utilizar barra de progreso de página.                    |
