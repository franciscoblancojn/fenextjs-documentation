# LayoutGridMenuTopLeft

El componente LayoutGridMenuTopLeft es un componente que muestra un layout con un menu arriba y a la izquierda.

### Ejemplo:

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=layout-layoutgridmenutopleft--index&viewMode=story" />

### Importación

Para importar el componente LayoutGridMenuTopLeft, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import {
  LayoutGridMenuTopLeft,
  LayoutGridMenuTopLeftProps,
} from "fenextjs/cjs/component/Layout/Grid/GridMenuTopLeft";
```

o

```tsx copy
import {
  LayoutGridMenuTopLeft,
  LayoutGridMenuTopLeftProps,
} from "fenextjs-component/cjs/Layout/Grid/GridMenuTopLeft";
```

### Parámetros:

| Parámetro           |    Tipo     | Requerido | Default | Descripción                                                   |
| ------------------- | :---------: | :-------: | :-----: | :------------------------------------------------------------ |
| menuLeft            | `ReactNode` |    No     |         | Menú izquierdo del diseño.                                    |
| menuLeftActive      |  `boolean`  |    No     |         | Indica si el menú izquierdo está activo en el diseño.         |
| menuLeftMovilActive |  `boolean`  |    No     |         | Indica si el menú izquierdo móvil está activo en el diseño.   |
| useHeaderButtonMenu |  `boolean`  |    No     |         | Utilizar botón de menú en el encabezado para mostrar el menú. |
| loader              |  `boolean`  |    No     |         | Indica si la página está actualmente en estado de carga.      |
| menuTop             | `ReactNode` |    No     |         | Menú superior del diseño.                                     |
| usePageProgress     |  `boolean`  |    No     |         | Utilizar barra de progreso de página.                         |

- Uso básico del componente LayoutGridMenuTopLeft

```tsx copy

  menuTop={
    <>
      <h3>It is Menu Top</h3>
    </>
  }
  menuLeft={
    <>
      <h3>It is Menu Left</h3>
    </>
  }
>
  <>
    <h1>It is Body</h1>
    <p>
      Lorem ipsum dolor, sit amet consectetur adipisicing elit. Aperiam
      laboriosam qui expedita nostrum facere consequuntur labore blanditiis
      perspiciatis ducimus est, iure dolorem nesciunt numquam modi hic, sit
      repellat at iste?
    </p>
  </>
</LayoutGridMenuTopLeft>
```
