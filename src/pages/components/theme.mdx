# Theme

El componente Theme es un componente que muestra un selectore de temas, claro, oscuro y automatico(toma el valor del navegador).

### Ejemplo

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=theme-theme--index&viewMode=story" />

### Importacion

Para importar el componente Theme, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import { Theme, ThemeProps } from "fenextjs/cjs/component/Theme";
```

o

```tsx copy
import { Theme, ThemeProps } from "fenextjs-component/cjs/Theme";
```
