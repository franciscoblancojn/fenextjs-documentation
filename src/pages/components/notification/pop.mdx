# NotificationPop

El componente NotificationPop es un componente que muestra una notificacion que se activa con el hook useNotification.

### Ejemplo:

<Iframe src="https://fenextjs-component-storybook.vercel.app/iframe.html?args=&id=notification-notificationpop--index&viewMode=story" />

### Importación

Para importar el componente NotificationPop, se puede hacer desde fenextjs o fenextjs-component

```tsx copy
import {
  NotificationPop,
  NotificationPopProps,
} from "fenextjs/cjs/component/Notification/Pop";
```

o

```tsx copy
import {
  NotificationPop,
  NotificationPopProps,
} from "fenextjs-component/cjs/Notification/Pop";
```

### Parámetros:

| Parámetro |            Tipo             | Requerido | Default | Descripción                                                      |
| --------- | :-------------------------: | :-------: | :-----: | :--------------------------------------------------------------- |
| \_t       |          `string`           |    No     |         | Propiedad de internacionalización.                               |
| typePop   |      `"top" \| "down"`      |    No     |         | Tipo de notificación emergente. Puede ser `top` o `down`.        |
| position  | `NotificationPositionProps` |    No     |         | Posición de la notificación emergente.                           |
| timeout   |          `number`           |    No     |         | Duración en milisegundos para mostrar la notificación emergente. |
| className |          `string`           |    No     |         | Clase personalizada para la notificación emergente.              |
